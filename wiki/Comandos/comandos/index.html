Comandos avançados

Aqui tem os principais comandos mais estruturados de forma bem resumida

/execute

Este comando é usado para escolher quem vai executar e onde e em quem os comandos vão ser executados, é composto de vários subcomandos, são 'blocos de comando' que formam esse comando. Sintaxe geral: /execute ❬subcomandos❭ ❬comando a ser executado❭

as : seleciona um executor do comando, sintaxe: /execute ... as ❬alvo❭ ... run ❬comando❭

at : seleciona em quem o comando vai ser executado, não mudando o executor, sintaxe /execute ... at ❬alvo❭ ... run ❬comando❭

in : seleciona em qual dimensão o comando vai ser executado, com as posições sendo dividas por 8 quando executado do Overworld para o Nether, e multiplicadas por 8 quando vice-versa, sintaxe: /execute ... in ❬dimensão❭ ... run ❬comando❭

positioned : seleciona o local de execução do comando, sintaxe: /execute ... positioned ❬X❭ ❬Y❭ ❬Z❭ ... run ❬comando❭

run : executa o comando em si, deve ser colocado no final, sinyaxe /execute ... run ❬comando❭

/scoreboard

Este comando é usado para gerenciar, adicionar e remover objetivos, pontuações, placares, times e tags.

Sintaxe geral scoreboard objetivos: ❬add|list|modify|remove|setdisplay❭ (❬slot❭) ❬objective❭ ❬argumentos❭

Sintaxe geral scorevoard jogadores: ❬add|enable|get|list|operation|remove|reset|set❭ ❬alvo❭ (❬objetivos❭) (❬argumentos❭) (❬pontuação❭)

Argumento "list" lista todos os objetivos e pontuações existentes.

Argumento "setdisplay" seleciona onde serão exibidas as pontuações podendo ser "list" ficando a pontuação na lista de jogadoes do servidor, "sidebar" ficando em uma tabela à direita do jogador e "bellowname" exibindo acima da cabeça do jogador.

Exemplo: /scoreboard ... setdisplay ❬tipo❭ ...

Argumentos "add", "remove" e "modify" servem para adicionar, remover e modificar um objetivo, tendo critério uma conquista ou ação específica.

Manipulação de pontuação

Argumento "add" serve para somar a pontuação de um jogador.

Argumento "set" serve para definir uma pontuação.
