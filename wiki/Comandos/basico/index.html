O que são e sintaxe

Um comando no minecraft é um texto que quando executado faz algo dentro no minecraft, para usa-los primeiro precisa ter os cheats (trapaças) ativados no seu mundo e, se for em um servidor, você precisa ser operador. Para executa-los pelo chat, coloca-se um "/" antes dos comandos, após isso, se digita os comandos. A sintaxe geral é / ❬comando❭ ❬argumentos❭, alguns comandos os argumentos pode ser:

/ ❬comando❭ ❬em quem vai ser executado❭ ❬local❭ ❬argumentos❭ ❬nbt❭ ;

Esses argumentos de cima são alguns dos mais usados. Os seletores de quem vai ser executado são:

@p : busca o jogador mais próximo do local de execução do comando

@r : executa em um jogador aleatório

@a : execute em todos os jogadores

@e : executa em todas as entidades

@s : "self", seleciona o executor do comando

Ainda tem parâmetros extras, são colocados com a sintaxe @#[❬seletor❭], os principais são:

r, rm : são os raio máximo e mínimo para buscar as entidades

type : seleciona a entidade específica a ser selecionada

x, y, z : é o local em que vai buscar as entidades, funciona melhor com o 'r'

l, lm : são os níveis de XP máximo e mínimo do(s) jogador(es) para que seja executado o comando

Alguns comandos básicos

/gamemode : altera o modo de jogo do jogador alvo; sintaxe: /gamemode ❬alvo❭ ❬modo de jogo❭

/gamerule : altera uma regra do mundo, sintaxe: /gamerule ❬regra❭ ❬true : false : número❭

/difficulty : altera a dificuldade do mundo, útil quando a dificuldade estiver travada, sintaxe: /difficuty ❬dificuldade❭

/give : dá um item para um jogador/entidade, sintaxe: /give ❬alvo❭ ❬item❭ ❬quantidade❭ ❬metadados❭ ❬NBT Data❭

/setblock : coloca um bloco em uma coordenada específica, sintaxe: /setblock ❬posição❭ ❬bloco❭ ❬metadados❭ ❬NBT Data❭

/summon : invoca uma entidade, sintaxe: /summon ❬entidade❭ ❬posição❭ ❬Rotação (opcional)❭ ❬NBT Data❭

/tp : teletransporta uma entidade, sintaxe : /tp ❬alvo❭ ❬destino (posição : quem)❭

/time set : coloca um horário específico no mundo, segundo os ticks ou período do dia, sintaxe /time set ❬horário❭

/time add : adiciona o tempo (em ticks) ao horário, sintaxe: /time add ❬tempo❭

/time query : diz qual é o horário do mundo em ticks, basta colocar /time query.

/weather : coloca um tempo (limpo, chuva ou tempestade) ao mundo, sintaxe /weather ❬tempo❭

/tell : manda uma mensagem para um ou mais jogadores.
